STEP 01
Subreddit + Community
X / Twitter
Read the
Community
Browser Use navigates the target subreddit and brand X mentions, reading posts sorted by "new" and "top." Full comment threads — the signal is often in the replies, not the post title.

Convex stores every post with a vector embedding. Semantic clustering groups "checkout keeps timing out" with "payment page crashes" even though the words differ, ranked by upvote mass and post frequency.

A single post with 1 upvote is noise. A cluster of 40 posts is a signal.

STEP 02
Classification
Claude
Bug or
Feature?
Claude reads the top cluster and classifies it. The label determines the entire downstream path.

🐛 Bug Report
Something broken that should work. "Checkout crashes on mobile," "cancellation email never arrives." Fix it and tell the community publicly.

✦ Feature Request
Something users want that doesn't exist yet. "Wish I could skip a week without calling," "no dark mode." Build it, then run an ad to turn the demand into acquisition.
Decision
Classification + rationale stored in Convex · routes to correct downstream branch

STEP 03
Daytona
Sandbox
Build the
Fix
Same process regardless of type — bug or feature.

Daytona spins up the target product codebase sandbox in <90ms with full filesystem, Git, and LSP. Claude writes the implementation with full repo awareness. Browser Use then navigates the locally running app inside the sandbox and clicks through the affected flow — verifying it actually works, not just compiles.

Agent opens a PR. Cubic reviews the diff repo-wide for cross-file issues and policy violations. Cubic fails → Claude revises in Daytona. Cubic passes → merge.
Output
UI click-tested by Browser Use in sandbox · PR opened · Cubic-approved · merged

STEP 04
Branching
Output
Post or
Advertise
🐛 If Bug
Browser Use replies to the original Reddit thread: "This has been fixed — update your app." Then posts a standalone announcement in the same subreddit community. No ad spend — the community that surfaced it hears about it first.

✦ If Feature
Mosaic generates a motion graphic from screenshots of the new feature. Dedalus Labs handles OAuth to Reddit Ads + X Ads. Browser Use places the ad targeted at the same audience that was asking for it. AgentMail issues tracked discount codes for sign-ups through the ad link.
Dual Path
Bug → public fix post on r/hellofresh · no spend  |  Feature → ad live on Reddit + X · tracked discount
